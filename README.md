# SRWEprep — CCNA 2 SRWE Practice Exam

A Practice exam tool for the **Cisco CCNA 2: Switching, Routing and Wireless Essentials (v7.02)** final exam.

---

## Features

- **171-question pool** drawn from real CCNA 2 SRWE exam topics
- **40–60 questions per test** (selectable), balanced across all 9 topic areas each run
- **Study Mode** — reveals the correct answer and explanation immediately after each selection
- **Test Mode** — simulates the real exam; all answers hidden until submission
- **All-or-nothing scoring** on multi-select questions (matches Cisco's actual exam policy)
- **Per-question explanations** for all questions, including ones that only had a topic reference in the source
- **41 exhibit images** with click-to-enlarge lightbox
- **Live timer** with color warnings at 60 and 75 minutes
- **Unanswered warning** before submitting — lists exactly which questions you skipped
- **End Early button** — score the test at any point without answering every question
- **Question flagging** — bookmark questions mid-test to revisit
- **Full answer review** after submission with correct/wrong/partial indicators and explanations
- **Score breakdown by topic** with a progress bar for each area
- **Dark mode** toggle
- **Adjustable font size** (A− / A+)
- **Dot navigation** — color-coded per question (green = correct, red = wrong, yellow = partially selected, grey = unanswered)

---

## Usage

1. Go to **https://aenlow.github.io/SRWEprep**
2. Choose **Study Mode** or **Test Mode**
3. Select how many questions you want (40–60)
4. Click **Start Practice Test**

---

## Scoring

- **Pass threshold:** 70%
- **Single-answer questions:** 1 point for correct, 0 for wrong
- **Multi-answer questions:** 1 point only if **all** correct answers are selected and no wrong ones (all-or-nothing, matching Cisco's real policy)
- Yellow dot on multi-select questions means you selected *some* correct answers but not all — still 0 points
